
# Apache Airflow

## 1. 배경 

일반적인 데이터 분석을 위한 과정으로는   
> 1. 어플리케이션에서 로그 및 사용자 정보 등의 데이터를 수집  
> 2. 수집된 원천 데이터에서 데이터를 추출하고 처리   
> 3. 처리되어 적재된 데이터를 바탕으로 데이터 사용하거나 결과를 저장  

해당 과정처럼 여러개의 TASK를 연결해서 수행해야 할 경우가 있다.    
단순하게 CRON + SHELL로 순차적으로 수행하는 것도 가능하지만, 재처리나 수행 결과에 따라 분기하는 등 조금 더 구조화 된 도구가 필요하다.   
이런 요구 사항 때문에 여러가지 툴이 개발되었는데, 대표적인 도구로는 Hadoop Ecosystem에 oozie가 있다.


## 2. Airflow 

Airflow는 oozie와 같은 데이터 워크플로우 관리 도구 이며, Airbnb에서 개발되었으며, 현재 Apache Incubator 프로젝이다.   
> 1. Python 기반으로 Task Code를 작성할 수 있어, 데이터 분석이나 머신러닝을 개발하는 엔지니어에게 친숙한 언어이다. 
> 2. Task를 **병렬**로 실행할 수 있습니다. 또한 한대에서 동작하는게 아니라 여러 머신에서 분산하여 수행 될 수 있는 장점을 가진다.
